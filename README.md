# React components for Web UI.

## Features:
- Storybook for UI development
- Emotion for write CSS in JS
- Jest Enzyme for testing
- Babel for transpiling
- Rollup for bundling

## Install

```sh
npm install
```

## Usage

### Auto fix rule

```sh
npm run lint
```

### Auto format code

```sh
npm run format
```

### Run Storybook

```sh
npm run sb
```

### Build lib

```sh
npm run build
```
